## 1. 어떤 프로젝트 인가? 
Final Project 진행 당시 특별한 아이디어가 없었던 저희 팀은
다들 한번씩 개발해보고 싶다던, **에어비앤비** 사이트를 모방하여 만들었습니다.

&nbsp;

당시 개발에 대해 좀더 잘 알던 팀원분들이 
- 숙소 생성 기능
- 지도 클러스터러 기능

등 다소 어려운 작업을 맡았고

&nbsp;

나머지 팀원분들은 그외 나머지 기능들을 분담해서 작업을 하였습니다.
